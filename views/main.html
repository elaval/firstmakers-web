First Makers


Making it simple for young makers!

Hardware

First Makers Board - shield for Aduino prebuilt with sensors and actuators. It is easily expandable with external modules (more sensors and actuators) that can be connected with common 3.5mm connector (like the ones used for connecting earphones). No need to solder, nor having especialised knowledge of electronics. Just need your curiosity and creativity to start connecting and controlling objects around you.

Software

First Makers Software - open source software that allows you to control your First Makers board in a few minutes. Visual programmable blocks let you capture sensor values and modifiy actuators. Very easy to give your first steps programming objects around you (and connect them to the Internet)!
