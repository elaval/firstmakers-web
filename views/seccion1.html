¡Simplificando los primeros pasos de jóvenes creadores!

Hardware

Placa First Makers - una placa para Arduino prearmada con sensores y actuadores. Es fácilmente ampliable con módulos externos (más sensores y actuadores) que se conectan con conectores comunes de 3.5mm (como los que se usan para conectar auriculares).

No hay necesidad de soldar componentes, ni tener conocimiento especializado de electrónica. Sólo se necesita tu curiosidad y creatividad para empezar a conectar y controlar los objetos que te rodean.


Las placas están en siendo probadas en nuestros laboratorios (fase de prototipo) y pronto estarán disponibles para el público.

Software

Software First Makers - software abierto que permite controlar tu placa First Makers en pocos minutos. Mediante bloques de programación visual puedes capturar valores de sensores y modificar los actuadores.

Es muy fácil dar los primeros pasos para programar los objetos que te rodean (¡y conectarlos a la Internet!)

Este software se ha construido sobre Snap4Arduino, un gran proyecto de código abierto desarrollado por Citilab en Cataluña.
